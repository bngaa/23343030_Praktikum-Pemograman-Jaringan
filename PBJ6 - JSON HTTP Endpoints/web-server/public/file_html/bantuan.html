Cara Menggunakan Aplikasi

1. Endpoint API Cuaca

Untuk mendapatkan data cuaca, gunakan endpoint berikut:

GET /infocuaca?address=<nama_lokasi>

2. Contoh Penggunaan

Misalnya untuk mencari cuaca di Jakarta:

GET /infocuaca?address=Jakarta

3. Response Format

Server akan mengembalikan JSON dengan format:

{
  "prediksiCuaca": "string",
  "lokasi": "string",
  "address": "string"
}

4. API yang Digunakan

  • Mapbox Geocoding API - Untuk mengonversi alamat ke koordinat
  • WeatherStack API - Untuk mendapatkan data cuaca berdasarkan koordinat

5. Troubleshooting

  • Pastikan koneksi internet stabil
  • Masukkan nama lokasi yang valid
  • Server berjalan pada port 5000
