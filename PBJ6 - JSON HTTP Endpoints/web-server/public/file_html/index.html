Selamat Datang

Aplikasi ini membantu Anda cek cuaca di lokasi manapun.

Cara Menggunakan:

Gunakan endpoint /infocuaca?address=nama_lokasi untuk mendapatkan data cuaca JSON.

Contoh Request:

GET /infocuaca?address=Jakarta

Contoh Response:

{
  "prediksiCuaca": "Info Cuaca: Clear. Suhu saat ini adalah 28 derajat...",
  "lokasi": "Jakarta, Indonesia",
  "address": "Jakarta"
}
